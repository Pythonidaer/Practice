Grid Nesting: Colors

CSS Critters!!!

I wanted to document this idea before going back to the course.

What I imagine doing is creating a grid-template with 3 columns:

  • The left and right would be (x)fr (variable fraction)
  • The middle would be a minimum-width of 600px

What I would like to achieve with this is experimenting with responsive web design before getting into media queries. I probably will make mistakes. I mean, it's only natural! I don't think I have to wait to get into media queries to start experimenting however. This is just an attempt of collapsing white space.
